allPossibleArrays(array, startIncopt, endExcopt, undefValueopt, non-null) → {Array.<Array.<number>>}

Returns all possible arrays that can be made using a base array, where each instance of `undefValue` is replaced with all values from `startInc` to `endExc`. For example: `allPossibleArrays([1, -1], 1, 4) => [[1, 1], [1, 2], [1, 3]] Another example: `allPossibleArrays([-1, 4, -1], 0, 2) => [[0, 4, 0], [0, 4, 1], [1, 4, 0], [1, 4, 1]]` It calls itself recursively for each instance of `undefValue`, then adds the arrays together, so it can be called on arrays of infinite length and infinite instances of `undefValue`.
